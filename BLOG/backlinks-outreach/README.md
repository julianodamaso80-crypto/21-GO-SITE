## 🚨 NÃO FAÇA (anti-patterns)

❌ Comprar tráfego (SparkTraffic, similarweb-traffic, etc) — não funciona em 2026
❌ Comprar backlinks em Fiverr/marketplaces — link farm = penalty
❌ PBN (Private Blog Network) — Google detecta via SpamBrain
❌ Trocar link em massa ("você linka pra mim, eu linko pra você") — pattern detectado
❌ Spam de comentários em blogs
❌ Forum signature spam
❌ Sub-domínios apontados pra mesmo IP (link farm)
